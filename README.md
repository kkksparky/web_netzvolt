# NetzVolt Static Marketing Website

Production-ready static marketing site for NetzVolt (DE/EN), built with pure HTML, CSS, and vanilla JavaScript for GitHub Pages.

## Quick Start
1. Open `index.html` in a browser.
2. No build step, no npm, no dependencies.

## Deployment (GitHub Pages)
1. Push all files to `main`.
2. Enable Pages in repository settings.
3. Source: `main` branch, root folder.

## Custom Domain
- `CNAME` contains `netzvolt.de`.
- Add GitHub Pages A records and enable HTTPS.

## Asset Setup
Place all final images in `./images/` with exact filenames referenced in HTML.

## Form Setup
Replace `YOUR-WEB3FORMS-KEY` in forms (hidden field) to activate submissions.
